Crea una aplicación Expo con Expo Router y NativeWind (Tailwind CSS para React Native) que muestre un portafolio de propiedades inmobiliarias de un broker. La aplicación está optimizada SOLO para web.

## Configuración Inicial

1. Expo SDK 51+
2. Expo Router para navegación
3. NativeWind v4 (Tailwind CSS para React Native)
4. Expo Image para imágenes optimizadas
5. Lucide React Native para iconos

## Comandos de instalación

```bash
npx create-expo-app@latest broker-portfolio --template tabs
cd broker-portfolio
npx expo install nativewind tailwindcss
npx expo install expo-image
npx expo install @expo/vector-icons
npm install lucide-react-native react-native-svg

4. components/ui/LocationCard.tsx

import { useState } from 'react';
import { View, Text, Pressable } from 'react-native';
import { Image } from 'expo-image';
import { Link } from 'expo-router';
import { ChevronRight } from 'lucide-react-native';

interface LocationCardProps {
  id: number;
  title: string;
  description: string;
  price: string;
  area: string;
  images: string[];
  href: string;
}

export default function LocationCard({ id, title, description, price, area, images, href }: LocationCardProps) {
  const [currentImageIndex, setCurrentImageIndex] = useState(0);
  const [isHovered, setIsHovered] = useState(false);

  const handleNextImage = () => {
    setCurrentImageIndex((prev) => (prev + 1) % images.length);
  };

  return (
    <Link href={href as any} asChild>
      <Pressable 
        className="flex-1"
        onHoverIn={() => setIsHovered(true)}
        onHoverOut={() => setIsHovered(false)}
      >
        <View className="overflow-hidden">
          {/* Imagen */}
          <View className="relative h-72 w-full">
            <Image
              source={{ uri: images[currentImageIndex] }}
              style={{ width: '100%', height: '100%', borderRadius: 16 }}
              contentFit="cover"
            />

            {/* Botón siguiente (visible en hover) */}
            {isHovered && (
              <Pressable
                onPress={(e) => {
                  e.stopPropagation();
                  handleNextImage();
                }}
                className="absolute right-2.5 top-1/2 -translate-y-1/2 bg-white/30 rounded-full p-1.5 w-7 h-7 items-center justify-center"
              >
                <ChevronRight size={20} color="#000" />
              </Pressable>
            )}

            {/* Indicadores */}
            <View className="absolute bottom-3 left-0 right-0 flex-row justify-center items-center gap-2">
              {images.slice(0, 5).map((_, index) => (
                <View
                  key={index}
                  className={`rounded-full bg-white-grey-3 ${
                    index === currentImageIndex ? 'w-1 h-1' : 'w-1.5 h-1.5'
                  }`}
                />
              ))}
            </View>
          </View>

          {/* Info */}
          <View className="flex-row justify-between items-center mt-2">
            <Text className="text-base font-bold text-white-total-black flex-1 mr-2" numberOfLines={1}>
              {title}
            </Text>
            <Text className="text-sm text-white-black">{area}</Text>
          </View>
          <Text className="text-sm text-white-grey2 mb-2" numberOfLines={1}>
            {description}
          </Text>
          <Text className="text-base font-bold text-white-total-black">{price}</Text>
        </View>
      </Pressable>
    </Link>
  );
}

Para ejecutar solo en web

npx expo start --web
Notas importantes para Expo Web
Usa expo-image en lugar de next/image - mejor rendimiento
Usa View y Text de react-native en lugar de div y span
Los estilos de flexbox funcionan con flex-row y flex-col
Las media queries (md:, lg:) funcionan con NativeWind v4
onHoverIn y onHoverOut solo funcionan en web
Los shadows necesitan definirse con style prop además de className
La ruta de acceso será: /blog/[cualquier-id]
Por ejemplo: /blog/demo-broker
